研究摘要

隨著全球失智症盛行率逐年攀升，落實早期預防與風險預測已成為臨床精準醫療的核心課題 。在醫療數位化高度普及的現況下，透過自動化手段獲取臨床電子病歷 (Electronic Medical Records, EMR)數據已成為現代醫療系統的基礎。為此，本研究利用美國國家阿茲海默症協調中心(National Alzheimer's Coordinating Center, NACC)的資料集，結合機器學習演算法與 RESTful API 輕量化架構，開發失智症風險預測系統。本研究旨在透過標準化介面實現數據與模型的即時串接，協助醫療人員即時評估患者在未來兩年內罹患失智症之風險，進而強化早期篩檢效率並優化臨床決策品質。

研究方法採用NACC資料集共23,795筆數據，經鏈式方程多重插補(Multiple Imputation by Chained Equations, MICE)插補缺失值，並運用Boruta演算法篩選關鍵特徵。模型建構比較了隨機森林(Random Forest, RF)、極限梯度提升(eXtreme Gradient Boosting, XGBoost)及羅吉斯回歸(Logistic Regression, LR)，並針對類別不平衡調整權重以提升預測召回率。系統實作採Flask建置RESTful API後端與Angular前端，整合Faster Whisper語音技術及在地化「十二生肖版連線測驗」，實現認知測驗的數位化與自動計分。

研究結果顯示，篩選出9項關鍵特徵的「均衡版隨機森林模型」表現最佳，召回率(Recall)達0.79、準確率(Accuracy)為0.85，顯著優於過往研究，能更有效識別高風險患者。經臨床場域驗證優化特徵後，確認了模型在實務上的可行性。本研究成功建立整合臨床數據流與預測模型的標準化系統，不僅提升篩檢效率，其RESTful API架構亦為未來跨機構數據共享奠定基礎。
