最佳模型：均衡版隨機森林 (Balanced Random Forest)

研究結果顯示，篩選出 9 項關鍵特徵的「均衡版隨機森林模型」表現最佳。透過對少數類別賦予較高權重，成功克服了醫療數據常見的類別不平衡問題。相較於 James et al. (2021) 等先前研究（Recall 約 0.27-0.28），本模型將 召回率大幅提升至 0.79，證明了特徵優化與權重調整策略在臨床篩檢應用上的顯著成效。

臨床意義解析

在失智症篩檢的臨床場域中，「偽陰性」(False Negative) 的代價遠高於「偽陽性」。若將潛在患者誤判為健康，將導致患者錯失藥物控制與生活干預的關鍵時期。

因此，本研究的核心目標並非單純追求最高的準確率 (Accuracy)，而是著重於提升召回率 (Recall)。實驗數據證實，透過 Boruta 篩選出的 9 項特徵（包含 CDR 相關指標），結合類別權重調整，能夠建構出一個高敏感度的防護網，協助醫師更敏銳地捕捉到早期的認知退化跡象。

RESTful API 實務應用

除了模型效能外，本研究成功驗證了透過 RESTful API 整合機器學習模型的可行性。利用 Flask 與 Angular 構建的前後端分離架構，展現了以下優勢：

  • 即時性： 臨床數據輸入後，API 可毫秒級回傳風險評估。
  • 擴展性： 標準化的 JSON 介面使得跨機構的電子病歷 (EMR) 系統能夠無縫串接。
  • 隱私性： 採用 JWT 驗證與無狀態設計，確保病患數據傳輸安全。
